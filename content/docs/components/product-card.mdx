ProductCard is the core shopping primitive in shoppablecn. It covers simple
catalog cards, denser comparison layouts, and quick-buy flows that slide the
variant picker directly over the card instead of navigating away from the page.

## Installation

`product-card` installs from the shoppablecn registry and automatically pulls
in the shared helpers and quick-options dependency it needs for overlay mode.

```bash
npx shadcn@latest add @shoppablecn/product-card
```

## Usage

The minimal setup is a `product` object and the `ProductCard` component. The
default rendering is the vertical layout shown below.

## Accessibility

- Keyboard navigation works for slideshow controls, action buttons, and overlay interactions.
- Icon-only actions expose accessible labels.
- Overlay mode manages focus and respects `prefers-reduced-motion`.
- Slide navigation remains available with arrow keys when the image region is focused.
